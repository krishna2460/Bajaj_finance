🚀 QUICK START (5 MINUTES)

1. Start MongoDB:
   - Windows: mongod
   - macOS: brew services start mongodb-community
   - Linux: sudo systemctl start mongodb

2. Terminal 1 - Start Backend:
   cd backend
   npm install
   cp .env.example .env
   npm run dev

3. Terminal 2 - Start Frontend:
   cd frontend
   npm install
   npm run dev

4. Open Browser:
   http://localhost:3000

5. Create Your First Graph:
   • Click "New Graph"
   • Add nodes and edges
   • Visualize and analyze
